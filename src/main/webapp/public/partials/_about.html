About

This application is written in Java. The backend is created with Spring Boot, Spring Data JPA and protected by Spring Security. The frontend is created with AngularJS except a little part for embedded maps.

I've blogged about my goals rewriting the old application from scratch here Developing a web application with Spring Boot, AngularJS and Java 8, the source code is available here github.com/michael-simons/biking2.

There's also a little companion app, a JavaFX 8 client obviously called BikingFX written in October 2014, after JavaOne. If you like, get started with JavaFX 8 right here: Developing a JavaFX REST client application from scratch.
The architecture and api of this service is documented here. The beautiful documentation is partially generated and then rendered using Spring REST Docs, a really clever solution, that ties api documentation to valid tests. The structural template is based on arc42, a standardized structure for construction, documentation and communication of software architectures.

Until early 2014 this application was Ruby and Sinatra based and consisted mainly of one big *.rb file. The blog post about the old application is still online here: Creating a self containing mvc application with Sinatra, together with the old source code.

If you like this application or have any questions, just drop me a line at misi@planet-punk.de.

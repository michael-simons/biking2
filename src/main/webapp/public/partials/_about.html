About

This application is written in Java. The backend is created with Spring Boot, Spring Data JPA and protected Spring Security. The frontend is created with AngularJS except a little part for embedded maps.

I've blogged about my goals rewriting the old application from scratch here Developing a web application with Spring Boot, AngularJS and Java 8, the source code is available here github.com/michael-simons/biking2.

Until early 2014 this application was Ruby and Sinatra based and consisted mainly of one big *.rb file. The blog post about the old application is still online here: Creating a self containing mvc application with Sinatra, together with the old source code.

If you like this application or have any questions, just drop me a line at misi@planet-punk.de.

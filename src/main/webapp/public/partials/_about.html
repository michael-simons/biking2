About

This application is written in Ruby and uses Sinatra and SQLite.

The chart is created using the amazing Highcharts JS library and the map is based on Google Maps.

I made heavily use of Sinatras inline templates and therefore the whole application is, although a full mvc based application, one single file still way under 1000 lines, including all model and css code.

All view code is written using Haml respectively Sass, including the auto-refreshing JavaScript Code, so no XHTML pollutes this source file.
